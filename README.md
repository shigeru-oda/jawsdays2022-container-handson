# jawsdays2022-container-handson
JAWS DAYS 2022でのコンテナハンズオン資料です

# 環境構築
## インストール
brew install go

## PATH
export GOPATH=$HOME/Go
export GOROOT=/usr/local/opt/go/libexec
export PATH=$PATH:$GOPATH/bin
export PATH=$PATH:$GOROOT/bin

## EXPORT
claat export <マークダウンファイル名>
